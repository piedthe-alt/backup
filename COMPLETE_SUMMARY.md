## 📱 Responsive Features

### Mobile-First Approach
```css
/* Base styles: mobile */
.card {
    border-radius: 12px;
    padding: 16px;
    font-size: 16px;
}

/* Tablet adjustments */
@media (min-width: 768px) {
    .card { font-size: 17px; }
}

/* Desktop adjustments */
@media (min-width: 1024px) {
    .card { font-size: 18px; }
}
```

### Responsive Grid
```html
<!-- Automatically responsive -->
<div class="row g-4">
    <div class="col-12 col-sm-6 col-lg-4 col-xl-3">
        <!-- Full width on mobile -->
        <!-- Half width on tablet -->
        <!-- 1/3 width on large screens -->
        <!-- 1/4 width on extra large -->
    </div>
</div>
```

### No Overflow Design
```css
/* Tables scroll horizontally on mobile */
.table-responsive {
    overflow-x: auto;
}

/* Images are fully responsive */
img {
    max-width: 100%;
    height: auto;
}

/* Cards auto-wrap */
.grid {
    grid-template-columns: repeat(auto-fill, minmax(250px, 1fr));
}
```

```html
    <option>Choose...</option>

```

### For New Pages
```html
<!-- 1. Include CSS link in <head> -->
<link href="{{ asset('css/modern-design-system.css') }}" rel="stylesheet">

<!-- 2. Use modern header pattern -->
<div class="card border-0 shadow-sm rounded-3 overflow-hidden mb-4" 
     style="background: linear-gradient(135deg, #2563eb 0%, #1d4ed8 100%); color: white;">
    <!-- Header content -->
</div>

<!-- 3. Use responsive grid -->
<div class="row g-4">
    <div class="col-12 col-md-6 col-lg-4">
        <!-- Content with auto-responsiveness -->
    </div>
</div>
```

### CSS Variables for Customization
```css
/* Use variables everywhere for consistency */
.my-component {
    color: var(--primary-600);
    padding: var(--spacing-md);
    border-radius: var(--radius-lg);
    box-shadow: var(--shadow-md);
    transition: all var(--transition-base);
}

/* Change 1 variable = affects entire app */
:root {
    --primary-600: #3b82f6; /* New blue */
}
```
